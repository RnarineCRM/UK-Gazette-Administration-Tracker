📥 Load Notices — Manual Paste

The Gazette's firewall now blocks automatic fetching, but a normal browser tab still works fine. Paste the day's list once, then the green panel below hands you each notice's detail page to click and paste. About a minute a day.

Other ways (advanced — only if the Copy button won't work)

Note: the old auto-collector bookmarklet and console script no longer work — the Gazette's firewall blocks background fetches. The manual paste in Steps 1–3 above is the reliable way.

Fully by hand (no script at all):

  1. Open the Gazette JSON list, select all (Ctrl+A), copy (Ctrl+C), paste in the box, click Load Into Table. If a busy day spills past one page, change results-page=2 in the URL and paste that too — duplicates are ignored.
  2. For each notice's full details, open https://www.thegazette.co.uk/notice/{ID}/data.jsonld, copy the JSON (it starts with {), paste, and click Load Into Table again. (Details fill in Date of Appointment, Administrators, addresses & Nature of Business, which the plain list leaves blank.) This data.jsonld address is used in place of the older data.json?view=linked-data one, which the Gazette rate-limits with an HTTP 429 after a few opens.
